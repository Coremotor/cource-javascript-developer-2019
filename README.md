 8. Разное.
 9. JavaScript в браузере. DOM.
 10. Web API.
 11. Клиент и сервер.
 12. jQuery.
 13. Современный JavaScript. ES 2015.
 14. Сборка проектов.
 15. React.
 16. Redux.
